Command List

  • ls - list directory
  • cd - change directory
  • touch - create empty file or update modification time
  • git add - adds a new file, or make changes to file
  • git commit - commits the changes to the file
  • git pull - checks for recent changes
  • git push - pushes changes to the webpage
  • git status - checks the status, recent changes
  • pwd - prints working directory
  • cd - change directory
  • cp - copies file/folder
  • mv - moves file/folder
