Command List

  • ls - list directory
  • cd - change directory
  • touch - create empty file or update modification time
  • git add - adds files to staging
  • git commit - commits added files to head
  • git pull - pulls changes from the server
  • git push - pushes changes to the server
  • git status - shows status of files, whether they are new/added/commited
  • git branch - shows avaliable branches in repo
  • git checkout - change current branch to selected one
  • git diff - show all merge conflicts
  • git merge - merge a branch onto your current branch
