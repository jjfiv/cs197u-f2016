Command List Changed

  • ls - list directory
  • cd - change directory
  • touch - create empty file or update modification time
  • git add - adds all of the files in the directory
  • git commit - records changes to the repository
  • git pull - Fetch from and and integrate with another repository or a local branch
  • clear - clears the screen
  • resert - resets the terminal display
  • pbpaste - paste clipboard contents
  • pbcopy - copies file contents to clipboard
  • ls - short listing
  • git push - update remote refs along with associated objects
  • git status - show the working tree status
