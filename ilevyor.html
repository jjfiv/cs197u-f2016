Command List

  • ls - list directory
  • cd - change directory
  • touch - create empty file or update modification time
  • git add - tells the tracker to care about the file
  • git commit - tells git to save the files that are being tracked locally
  • git pull - pulls changes from remote repository to the local ones
  • git push - pushes your local changes to the remote repository
  • git status - tells you about the current state of the tracked files
  • pwd - print the working directory
  • ls - list the current directory
  • cat - print what is said in a file
  • cowsay - make a dog that says things
