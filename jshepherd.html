OMGHI List

  • ls - list directory
  • cd - change directory
  • touch - create empty file or update modification
  • git add - add one or more files to staging
  • git commit - commit changes to head
  • git pull - fetch and merge changes on the remote server to working directory
  • git push - send changes to master branch
  • git status - list the files changed, and those you still need to add/commit
  • pwd - print working directory
  • ls - list files and directories in current working directory
  • cat - prints the file
  • chmod - change permissions of a file
  • mv - move a file
